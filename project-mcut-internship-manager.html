Context
Problem → Solution
The Problem
  • Data integrity at risk — shared Google Sheets exposed records to accidental deletion or tampering across departments.
  • No real-time tracking — student application status was only updated manually upon arrival confirmation.
  • Communication gaps — pre-arrival emails and exit reminders were sent ad hoc, causing delays and missed steps.
  • Exit process bottlenecks — students missed feedback submissions; staff had to chase completions manually.
The Solution
  • Centralised app — single source of truth for all internship student data, accessible to authorised staff only.
  • Automated workflows — Power Automate triggers comms at key milestones: pre-arrival, check-in, exit, and completion.
  • Digital exit process — students submit forms digitally; staff close cases in one click.
  • Live reporting — Power BI dashboards give leadership instant visibility into program performance.

Key Features
